# Config Directory

This directory is reserved for MCP server configuration files.

## Current Usage

The MCP server configuration is currently stored in Claude Code's config directory:
- Location: `~/.config/claude/config.json` (or equivalent for your system)
- See [README.md](../README.md#installation--setup) for configuration instructions

## Future Use

This directory may be used for:
- Example configuration files
- Environment-specific MCP server settings
- Shared configuration templates

## Example Configuration

See `claude-config-example.json` in the root directory for an example MCP server configuration.
